# textonly.website

A collection of personal websites or blogs that are lightweight and text-only.

## About

This project hosts a curated list of minimal, text-focused websites and blogs. The goal is to showcase content that prioritizes readability and accessibility over visual complexity.

## Contributing

Submit a website through the form on the main page.

## Usage

Visit [textonly.website](https://textonly.website) to browse the collection of text-only websites.
